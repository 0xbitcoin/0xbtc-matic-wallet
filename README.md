# 0xBTC Matic Wallet

A web3 wallet for 0xBTC that works with the Matic and Ethereum networks.

This dapp is deployed at Https://wallet.0xbtc.io


### Development commands
```
npm install
npm run dev
```

### Packaging commands
```
npm run build
npm run server
```
